# File Management Landing Page

A modern and responsive landing page built using pure HTML, CSS, and JavaScript to showcase a file management system and its key features.

## Features
- Responsive design (mobile-friendly)
- Clean and minimal user interface
- Lightweight and fast performance (no frameworks)

## Tech Stack
- HTML5 — semantic structure
- CSS3 — styling and layout (Flexbox/Grid)
- JavaScript (Vanilla) — interactivity and DOM manipulation

## Installation
1. Clone this repository
2. Open `index.html` in your browser
